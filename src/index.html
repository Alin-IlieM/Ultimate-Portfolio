Project Robofriends

This is a simple project that I've created in React through Andrei Neagoie's "The Complete Web Developer in 2023: Zero to Mastery" course. It basically takes a list of usernames from a JSON placeholder and generates unique robots using the robohash website. It has a custom font title, it uses "tachyons" for the styling, it's scrollable and has a working searchbox.
